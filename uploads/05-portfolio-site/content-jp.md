## About

東京を拠点とするソフトウェアエンジニアです。キャリアはグラフィックデザイナーとしてスタートしました。中国美術学院を卒業後、約2年間パッケージデザインやポスター、ブランドビジュアルの制作に携わり、その後プログラミングを独学でエンジニアに転身しました。

現在はアイフル株式会社で約3年間、金融サービス向けのフルスタックWeb開発を担当しています。フロントエンド（Vue.js）、バックエンド（Java / Spring Boot）、インフラ（AWS）を横断し、フレームワークのメジャーバージョン移行やサーバーレスシステムの0→1設計を主導してきました。

今後はAIエンジニアリングへのピボットを目指しています。LLMやAIエージェントを活用したプロダクト開発に、Web開発とクラウドの実務経験を土台として取り組んでいきたいと考えています。

中国語（ネイティブ）、日本語（JLPT N1）、英語（TOEIC 905）のトリリンガルです。

## Engineering Projects

### DualRead — 英語学習者向けChrome拡張機能
中国語話者のための英語学習Chrome拡張機能。Webページ上の英語テキストを選択するだけで即座に中国語翻訳を表示し、未知の単語を個人の単語帳に保存、保存した単語は訪問するすべてのページで自動ハイライト表示。アカウント不要、バックエンドなし、テレメトリなし — プライバシーファーストの設計。

**タグ:** TypeScript · React 19 · Chrome MV3 · Vite · CSS Variables
**リンク:** [GitHub](https://github.com/Foxsunshine/DualRead) · Chrome Web Store（v2.0）
**ハイライト:**
- 個人プロジェクトとしてゼロから設計・実装（0→1）
- アーキテクチャ: content script（ハイライトエンジン）+ background service worker（翻訳プロキシ、メッセージルーター）+ side panel（React UI）
- `chrome.storage.sync`によるデバイス間単語帳同期、`chrome.storage.session`による翻訳キャッシュ
- 主な機能: ドラッグ選択翻訳、単語境界スナッピング、Shadow DOMページ内バブル、Anki用CSVエクスポート、フローティング学習モード切替
