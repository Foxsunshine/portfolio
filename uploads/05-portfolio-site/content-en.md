## Engineering Projects

### DualRead — Chrome Extension for Language Learners
A Chrome side-panel extension for Chinese speakers learning English. Select any English text on any webpage to get instant Chinese translation, save unknown words to a personal vocab list, and see them auto-highlighted on every page you visit. No account, no backend, no telemetry — privacy-first design.

**Tags:** TypeScript · React 19 · Chrome MV3 · Vite · CSS Variables
**Links:** [GitHub](https://github.com/Foxsunshine/DualRead) · Chrome Web Store (v2.0)
**Highlights:**
- Built from scratch as a solo project (0→1)
- Architecture: content script (highlight engine) + background service worker (translate proxy, message router) + side panel (React UI)
- Vocab sync across devices via `chrome.storage.sync`; translation cache in `chrome.storage.session`
- Features: drag-select translation, word-boundary snapping, Shadow DOM in-page bubble, CSV export for Anki, floating learning-mode toggle
